
# 路上活動でよく聞かれることへの回答例

## 線引きするのは傲慢だ

## どこに意識の有無の線があるの？動物だって意識がないものはいるんじゃない？
知らない。
でも、それを知らなくても、豚や、牛や、鶏や魚が、人間と同様に、意識を持つことは分かりきっている。
彼らは苦しんでいる。こうした動物たちを、今後どうするべきかは、線がどこかを明確にしなければわからないことなの？
逆に線が無いとわからないなら、何故人間を殺したり食べたりしてないんでしょうかね？

## 植物だって生きている
veganの主張は「全ての命は平等」ではない。
苦しみを自分のこととして感じることができるものは、危害を加えるのをやめましょう、という主張です。

#### 植物にも意識はあると聞いた
あなたは、最先端の植物の研究でもしてるんですか？
それとも、結構前に出た、埼玉大の教授の論文のことですか？
植物を擬人化するような記事を書いたと、教授本人が認めている。
それ以外にも、植物は叫ぶ、 だとか、目を引く見出しの記事はある、広告収入を稼ぐための。

#### 本当に植物には意識がにないんでしょうね？
仮に「植物に意識がある」と仮定すると、説明がつかないことだらけです。動物は、暑い！と思ったら日陰に移動したり、空腹を感じたらなにか食べ物を探したり、痛みや苦しみを感じて、動きまわって、問題を解決する。
植物は動物のように柔軟に動き回れません。。仮に痛いと「感じ」たとしてもですよ、避けられないんです。意識はあるけど、動けない。ということになる。
それを踏まえると、意識はない、と考えるほうが遥かに合理的です。

#### だけど、覆って、植物にも意識があったら？
* 動物は苦しむことがわかっている。
* 植物は苦しむ可能性を説明できるものはない。
というときに、苦しむことがわかり切ってる動物に危害を加えよう！と、思うのですか？苦しむ可能性がほぼ無いとわかっているもので代替しようとは思いませんか？

#### だけど、覆って、植物にも意識があったら？２
牛やブタや鶏は、植物を食べます。
その牛やブタや鶏を食べるということは、間接的に植物を「犠牲」にします。
直接植物を取るよりも、肉を経由すれば、遥かにたくさんの「犠牲」があります。
本気で植物の犠牲を憐れむのであれば、動物を食べないことだ。

## 弱肉強食では？
「これは、弱肉強食である。ゆえに仕方がないのだ」ということですか？
人は、弱肉強食に盲目的に従うのではなく、弱者を助けることができる生き物です。
たとえば、法律って弱肉強食になってますか？

自然界では、適応できない個体は自然淘汰される。積極的に親から殺されることもある。
自然は弱肉強食だとしても、自然を手本にする理由はない。
